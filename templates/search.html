The Python script search_engine.py applies latent semantic indexing and singular value decomposition to implement a toy search engine. To run the script, you'll need Python 2.6 or greater and the numpy library. Also the Pickle file lsiData.pkl must be in the same directory as the script or the path to a similarly encoded word-document count file must be provided on the command line. See the README for details.
